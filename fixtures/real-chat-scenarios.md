# Real Chat Scenarios

## Start

用户：开波，今天练什么？我时间有限，肩和踝康复优先。  
AI：今天做上肢拉 + 核心 + 肩踝康复短版……

## Log Set

用户：下拉第一组 26kg×10，感觉更多是手臂累，背没感觉。  
AI：OK，先不加重量，下一组先肩胛下压再拉……

## Amend Set / Adjust

用户：第二组还是没感觉，想加重量试试？  
AI：先别加重量，换成同重量更难的方式……

## Swap Exercise

用户：推还是不舒服，侧平举会疼。今天能不能不练推？  
AI：可以，今天完全不练推……

## Next Set

用户：引体我现在做 8/6/5/4/3，后面掉得快。下一组咋办？  
AI：下一组别硬凑次数，用质量补足……

## End Session

用户：今天做完了，腰有点酸但不是刺痛。要不要继续加练？  
AI：今天到这就停，现在做 5 分钟收尾……

## Pre Workout Decision

用户：晚上6:40，很饿但想做4×4。先吃晚饭还是先练？  
AI：先吃训练前小加餐 200–400 卡……

## Conditioning Protocol

用户：我4×4心率上不去，你之前 spm 是不是太低？  
AI：4×4 看心率和呼吸，不看 spm 本身……
